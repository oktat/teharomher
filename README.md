# Háromszög területe

Háromszög területének számítása az háromszög oldalaiból.

Tesztelésre szánt verzió.

## Online

* [https://szit.hu/m/teharomher](https://szit.hu/m/teharomher)

## Telepítés

```cmd
git clone https://github.com/oktat/teharomher.git
cd teharomher
npm install
npm run dev
npm start
```

## Build

```cmd
npm run build
```
